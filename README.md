# 💎 SODO Kids Academy

Aplicație web de aritmetică mentală pentru copii (5–12 ani).

## Jocuri

- **🧮 Abac virtual** — abac soroban interactiv: joacă liber, construiește numărul cerut sau citește numărul de pe abac
- **⚡ Flash Anzan** — numerele apar rapid pe ecran, copilul le adună în minte
- **⏱️ Exerciții cronometrate** — adunare, scădere, înmulțire și împărțire contra cronometru
- **🧠 Memorie numerică** — privește numărul, ține-l minte și scrie-l din memorie

## Niveluri

5 niveluri de dificultate, de la **Începător** (5–6 ani) la **Expert** (11–12 ani). Toate nivelurile sunt mereu deblocate; stelele și scorurile sunt valabile pe durata sesiunii.

## Rulare locală

```bash
npm install
npm run dev
```

## Publicare

Aplicația se publică automat pe GitHub Pages la fiecare push pe `main` (vezi `.github/workflows/deploy.yml`).
